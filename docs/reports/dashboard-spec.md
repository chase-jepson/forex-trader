# Dashboard Spec

The v1 dashboard uses Streamlit and reads from the same service/repository layer as the trading engine.

## Live Market

Shows quote, spread, session state, latest signal, risk decision, open position, stop loss, take profit, hold timer, and PnL.

## Trade Reviews

Shows completed trades, blocked trades, rule explanations, mistake tags, and improvement hypotheses.

## Reports

Shows daily and weekly metrics from persisted data.
